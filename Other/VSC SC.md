
**1. Essential Navigation**

- **Quick Open File:** `Ctrl` + `P` (Win/Linux) or `Cmd` + `P` (Mac). Search by filename to jump between files instantly.
- **Go to Symbol:** `Ctrl` + `Shift` + `O` (Win/Linux) or `Cmd` + `Shift` + `O` (Mac). Jump directly to a specific function or variable in the current file.
- **Go to Definition:** `F12`. Jump to the source code of the function or variable under your cursor.
- **Go to Line:** `Ctrl` + `G`. Type a line number to jump there directly.
- **Switch Tabs:** `Ctrl` + `Tab` (Cycles recent files) or `Ctrl` + `PgUp`/`PgDn` (Move left/right through tabs). 

**2. Efficient Editing & Selection**

- **Select Current Word:** `Ctrl` + `D` (Win/Linux) or `Cmd` + `D` (Mac). Pressing it repeatedly selects the next matching occurrence, creating **multiple cursors**.
- **Select All Occurrences:** `Ctrl` + `Shift` + `L` (Win/Linux) or `Cmd` + `Shift` + `L` (Mac). Adds cursors to every match of the selected text in the file.
- **Expand Selection:** `Shift` + `Alt` + `→`. Smartly expands your selection based on code blocks (e.g., selecting inside brackets, then the whole function).
- **Delete Line:** `Ctrl` + `Shift` + `K` (Win/Linux) or `Cmd` + `Shift` + `K` (Mac).
- **Move Line Up/Down:** `Alt` + `↑` / `↓` (Win/Linux) or `Option` + `↑` / `↓` 
**3. Layout & Terminal Management**

- **Toggle Sidebar:** `Ctrl` + `B` (Win/Linux) or `Cmd` + `B` (Mac).
- **Toggle Terminal:** `` Ctrl + ` `` (Backtick).
- **Split Editor:** `Ctrl` + `\` (Win/Linux) or `Cmd` + `\` (Mac). Opens the current file in a side-by-side view.
- **Focus Explorer:** `Ctrl` + `Shift` + `E`. Moves focus to the file tree so you can navigate folders with arrow keys. 

**4. Productivity Pro-Tips**

- **Command Palette:** `Ctrl` + `Shift` + `P` (Win/Linux) or `Cmd` + `Shift` + `P` (Mac). Type any command (e.g., "Format Document") if you forget its shortcut.
- **Rename Symbol:** `F2`. Renames a variable or function everywhere it is used in your project.
- **Format Document:** `Shift` + `Alt` + `F` (Win/Linux) or `Shift` + `Option` + `F` (Mac).
